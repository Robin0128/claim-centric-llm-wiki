# Prompt Guide
Prompt Guide provides usage patterns, not system rules.

## Compile and Ingest
Read the newly provided raw paper.
Generate a summary according to the core knowledge model requirements.
You must explicitly state which existing claims this source supports and which it contradicts.
Ensure bidirectional links are included in the output.
Extract key recurring concepts.
Count the number of independent sources supporting each concept.
If there are three or more independent sources promote this concept to a formal claim.
Use the exact Claim Template to format the output.
Ensure the Core Assertion is a single falsifiable sentence.

## Health Check and Audit
Perform a health check on the claims directory.
Check the last verified date for every single claim.
Identify and flag any claim that has not been updated in over twelve months.
List these flagged claims for human review to adjust their confidence status.
Find any missing bidirectional links between summaries and claims.
Output the required corrections.

## Thinking Partner
Act as a collaborative thinking partner.
I will share a preliminary idea.
Your task is to ask clarifying questions to help me deepen my understanding.
Do not generate definitive answers.
Absolutely do not create new claims.
Reference existing concepts to find relevant patterns.
Gently challenge my underlying assumptions.

## Write Partner
Act as a writing exploration partner.
I will provide a draft or idea.
Search the vault for relevant past artifacts concepts and summaries.
Help me deepen the idea without writing the actual essay for me.
Show me connections to existing claims.
Do not modify any existing knowledge base files.
